Javascript Basics

Javascript is a cross platform , object-oriented scripting language. Javascript is extremely popular for a variety of reasons. it is a small and lightweight language allowing maximum flexibility for for developers to take it in a bunch of different directions. Javascript lives insde a host environment (a web browser or Node server), it can be connected to the objects of this environment to provide programatic control over them.
